# TeslaCam Studio

TeslaCam Studio 是一个用于查看特斯拉行车记录仪视频的桌面应用。它可以读取 TeslaCam 文件夹中的视频片段，自动整理时间线，并同步播放多个摄像头视角，帮助你更方便地回看行车记录、哨兵模式片段和事故前后的画面。

项目的目标是做一个清晰、直观、专注本地查看体验的 TeslaCam 工具。你不需要上传视频，也不需要把素材交给第三方服务处理，所有文件读取、视频播放和遥测解析都在本机完成。

## 主要功能

- 打开本地 TeslaCam 文件夹
- 自动扫描并识别按分钟切分的 MP4 视频片段
- 根据文件时间构建连续时间线
- 文件夹包含多天素材时，按日期拆分时间线并支持切换预览
- 同步播放多个摄像头视角
- 点击下方预览画面，将对应视角切换到主画面
- 支持拖动进度条快速定位到指定时间
- 支持一键快进或倒退 15 秒
- 支持导出当前主视角截图，并在图片中写入时间水印
- 支持导出当前主视角的一段 MP4 视频，并在视频中写入时间水印
- 在画面和信息栏中显示车辆状态
- 从 front 主视角视频中读取遥测信息
- 支持本地桌面端运行，不上传视频文件

## 支持的摄像头视角

TeslaCam Studio 会尽量识别常见的 TeslaCam 文件命名，并将同一时间点的视频归并为一组。

当前支持的视角包括：

- Front，前视摄像头
- Rear / Back，后视摄像头
- Left Repeater，左翼子板摄像头
- Right Repeater，右翼子板摄像头
- Left B-Pillar，左侧 B 柱摄像头
- Right B-Pillar，右侧 B 柱摄像头

## 遥测信息

如果视频中包含可读取的遥测数据，TeslaCam Studio 会在播放时显示车辆状态信息，包括：

- 实时时间
- 车速
- 油门开度
- 方向盘角度
- 刹车状态
- 左右转向灯状态
- 挡位
- Autopilot 状态
- GPS 位置
- 航向
- 三轴加速度

遥测信息只从 `front` 视角视频中读取，并以 front 视频作为同步播放的主时钟。部分 TeslaCam 视频可能不包含这类数据；这种情况下视频仍然可以正常播放，只是遥测区域会显示为空或不可用。

## 文件夹和文件命名

TeslaCam 视频通常按分钟切片保存，同一分钟可能包含多个摄像头视角。例如：

```text
2026-01-01_12-34-00-front.mp4
2026-01-01_12-34-00-left_repeater.mp4
2026-01-01_12-34-00-right_repeater.mp4
2026-01-01_12-34-00-back.mp4
```

TeslaCam Studio 会根据文件名中的时间戳和视角名称，把这些文件组织成一条连续时间线。

## 使用方式

启动应用后，点击“打开文件夹”，选择包含 TeslaCam 视频的目录。程序会自动扫描文件夹中的 MP4 文件，并在界面中显示可播放的时间线。

播放时可以：

- 点击播放 / 暂停按钮控制视频
- 点击快进 / 倒退按钮跳转 15 秒
- 拖动进度条跳转时间
- 当文件夹包含多天素材时，通过顶部日期选择器切换当天时间线
- 点击下方任意小画面切换主视角
- 查看右侧信息栏中的车辆状态

## 隐私说明

TeslaCam Studio 是本地桌面应用。视频文件和遥测数据都在你的电脑上处理，不会上传到云端，也不会发送到第三方服务器。

## 下载和发布

项目会通过 GitHub Releases 发布桌面端安装包。发布版本通常会包含 Windows、macOS 和 Linux 构建产物。

如果你从源码运行或自行打包，请确保已经安装 Node.js 和项目依赖。

## 项目状态

TeslaCam Studio 仍在持续完善中。不同车型、不同软件版本生成的视频文件可能存在差异，摄像头命名和遥测数据也可能有所不同。如果遇到某些视频无法识别或遥测显示异常，可以保留原始文件结构并提交问题反馈。
